Why This Project?

Understanding the purpose behind building this website

Purpose of the Website

This project was created to practice building a responsive website using HTML, CSS, and Bootstrap. The goal is to understand how modern websites are structured and styled while using a framework to speed up development.

What I Learned

HTML Structure

Learned how to properly structure a webpage using semantic elements like header, main, section, and footer.

Bootstrap Integration

Used Bootstrap's grid system, navbar, cards, and utility classes to create a clean, responsive layout.

Custom Styling

Applied custom CSS to match the playful theme with Comic Sans and a cat-themed header background.
